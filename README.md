## How to build?
* Only tested on Linux at the moment

### install rust
* https://rustup.rs/

### install wasm-pack
* https://github.com/rustwasm/wasm-pack

### install NodeJS
#### install node lts
* https://github.com/nvm-sh/nvm

#### install yarn
* `npm i -g yarn`

### Build
* `cd muscad/muscad-wasm`
* `wasm-pack build`
* `cd ../muscad-wgl-ui`
* `yarn`
* `yarn dev`
